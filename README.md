Website Institucional

Projeto de website institucional desenvolvido para apresentar informações, serviços e canais de contato da organização de forma moderna, acessível e responsiva.

Objetivo

Criar uma presença digital profissional, facilitando o acesso às informações da instituição e melhorando a comunicação com o público.

Funcionalidades

- Design responsivo para dispositivos móveis e desktop
- Navegação simples e intuitiva
- Seções informativas sobre a instituição
- Formulário de contato
- Integração com redes sociais
- Layout otimizado para experiência do usuário

Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
